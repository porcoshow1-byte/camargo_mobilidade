# Camargo Mobilidade

Plataforma de mototáxi e entregas 24h para Ourinhos - SP.

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS
- Supabase (Auth + Database + Storage)
- Capacitor (Android)
- Mapbox / Leaflet / Google Maps
- Mercado Pago (Pagamentos)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Hospedado no Vercel com SPA rewrites.
